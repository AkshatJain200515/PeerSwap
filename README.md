🎓 PeerSwap: Peer-to-Peer Academic Collaboration
Student Name: Akshat Jain
Module: B203 Software Engineering

PeerSwap is a MERN-stack web application designed to bridge the "Metadata Vacuum" in student collaboration. It allows students to find study partners across a directory of 1,000+ profiles using a custom Identity Resolution Algorithm.

🛠 Project Structure
/backend: Node.js & Express API (Port 5000)

/frontend: React + Vite UI (Port 5173)

docker-compose.yml: Orchestration for the full-stack environment.
